# Avalonia_Editor

### 介绍
#基于Avalonia的文本编辑器

 
## (下载release体验)[https://gitee.com/SuperWindcloud/avalonia_-editor/releases/download/1.1.1/SuperEdit.exe]


## 功能
- 打开文件
- 保存文件
- 打开文件所在目录
- 复制、粘贴、剪切,全选,撤销,重做,查找,替换
- 字体、字号、颜色
- 自动换行
- 自动补全
- 语法高亮
- 行号
- 自动保存
- 主题切换
- 多语言支持
